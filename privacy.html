Bio-Scheduler プライバシーポリシー

最終更新日: 2026年6月15日

Bio-Scheduler（以下「本アプリ」）は、バイオリズム・予定・天気・祝日を1画面で確認できる Webアプリ（PWA）です。本ポリシーは、本アプリが扱う情報とその取り扱いを説明します。 本アプリはサーバーを持たず、入力データは利用者の端末（ブラウザ）内にのみ保存されます。

アプリ名: Bio-Scheduler ／ サポートメール: eins56@gmail.com ／ 公開URL: https://eins2019.github.io/bio-scheduler/

1. 取得・保存する情報と目的

  • 入力データ（生年月日・予定・各種設定）: バイオリズム計算と予定表示のために、 利用者の端末のブラウザ保存領域（localStorage）にのみ保存します。外部サーバーには送信しません。
  • 位置情報: 天気予報の取得に使用します。端末の位置情報を許可した場合のみ取得し、 緯度・経度を天気サービス（Open‑Meteo）へ送信して予報を取得します。位置情報を保存・追跡しません。 地域を手動指定した場合は位置情報を使用しません。
  • Googleアカウント情報（任意の連携時のみ）: 利用者がGoogleカレンダー連携を行った場合、 読み取り専用スコープ（calendar.readonly）でカレンダーの予定を取得し、 表示のために端末内に一時キャッシュします。サーバーには送信・保存しません。予定の作成・変更・削除は行いません。

2. 第三者への提供

本アプリは利用者の個人データを第三者に販売・提供しません。機能提供に必要な範囲で、 天気サービス（Open‑Meteo）へ位置情報、Google APIへ認証要求を送信するのみです。

3. Google ユーザーデータの取り扱い（Limited Use）

本アプリがGoogle APIから受け取る情報の使用および転送は、 Google API Services User Data Policy （Limited Use 要件を含む）に準拠します。Googleカレンダーの予定データは、画面表示の目的にのみ使用し、 広告等の目的には利用せず、第三者へ転送しません。

Bio-Scheduler's use and transfer to any other app of information received from Google APIs will adhere to the Google API Services User Data Policy, including the Limited Use requirements.

4. データの削除・連携の解除

  • 本アプリ内のデータは、ブラウザの「サイトデータ／履歴の削除」で消去できます。
  • Googleカレンダー連携は、アプリ内の「連携済み」ボタンから解除できます。
  • 付与したアクセス権は Googleアカウントのアクセス管理 からいつでも取り消せます。

5. Cookie・トラッキング

本アプリはCookieを使用せず、解析・広告・第三者トラッキングを行いません （設定保存はブラウザのlocalStorageのみ）。

6. 免責事項

バイオリズムは娯楽・参考を目的とした表示であり、科学的な正確性を保証するものではありません。 天気予報・祝日・カレンダー情報の正確性・完全性についても保証しません。本アプリの利用により生じた いかなる損害についても、運営者は責任を負いません。
